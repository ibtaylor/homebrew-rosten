# homebrew-rosten #
Homebrew formulae for Edward Rosten's ([https://github.com/edrosten](https://github.com/edrosten)) computer vision and robotics libraries.

## To use: ##

`brew tap randvoorhies/rosten` and then `brew install <formula>`

For example:

`brew install TooN`

`brew install gvars`

`brew install libcvd`
